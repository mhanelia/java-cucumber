# java-cucumber

Material utilizado para realizar o curso "Aprenda BDD com Cucumber em JAVA" ministrado pelo Francisco Wagner Costa Aquino disponível na Udemy

## Instalação:

1- no desenvolvimento do projeto foi utilizado:
    1.1- Java SDK 8U291
    1.2- JRE 1.8

2- clone o diretório em uma pasta:

```bash
git clone https://github.com/mhanelia/java-cucumber.git
```

3- para executar os testes, basta acessar a pasta e executar `mvn test`:

```bash
java-cucumber> mvn test
```
